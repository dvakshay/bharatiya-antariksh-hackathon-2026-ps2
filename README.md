# Bharatiya Antariksh Hackathon 2026 - PS2

## Problem Statement

Generative AI-Based Cloud Removal for LISS-IV Satellite Images

## Objective

Satellite images are often obscured by clouds, making Earth observation difficult. The objective of this project is to develop an AI-based system that reconstructs cloud-free satellite imagery from cloudy observations.

## Planned Workflow

1. Dataset Collection
2. Data Preprocessing
3. Exploratory Data Analysis
4. Model Development (U-Net)
5. Model Evaluation
6. Streamlit Deployment
